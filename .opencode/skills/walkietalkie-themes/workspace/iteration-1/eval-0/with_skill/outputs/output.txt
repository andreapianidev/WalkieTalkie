FILE 2: ThemeAnimatedPack.swift — Add metadata registration block
================================================================================

Insert the following block at the end of `register(into:)`, after the arctic MARK
(lines 81-98):

        // MARK: - Tropical (NEW Pro V4)

        dict[.tropical] = ThemeMetadata(
            accentColor: Color(red: 0.0, green: 0.78, blue: 0.68),
            displayNameKey: "theme.name.tropical",
            iconName: "tree.fill",
            isProLocked: true,
            soundPackID: "synth",
            backgroundPrimary: Color(red: 0.03, green: 0.10, blue: 0.08),
            surfaceColor: Color(red: 0.06, green: 0.18, blue: 0.14),
            textPrimary: Color(red: 0.90, green: 0.98, blue: 0.94),
            textSecondary: Color(red: 0.62, green: 0.82, blue: 0.70),
            backgroundStyle: .gradient,
            gradientTop: Color(red: 0.02, green: 0.08, blue: 0.07),
            gradientBottom: Color(red: 0.05, green: 0.14, blue: 0.11),
            cardStyle: .elevated,
            accentGlow: Color(red: 0.0, green: 0.78, blue: 0.68).opacity(0.20)
        )

Design rationale:
  - Accent (0.0, 0.78, 0.68): bright turquoise → shallow tropical water
  - Gradient dark teal (top 0.02/0.08/0.07 → bottom 0.05/0.14/0.11):
    deep ocean → sunlit shallows, 10-20% difference as per guidelines
  - Surface (0.06, 0.18, 0.14): ~2x brighter than bgPrimary → .elevated card
  - TextPrimary (0.90, 0.98, 0.94): warm white with green tint → sand/foam
  - TextSecondary (0.62, 0.82, 0.70): ~65% of primary → creates depth
  - Icon "tree.fill": palm tree silhouette → tropical beach identity
  - Font nil: clean modern system font → fresh tropical feel
  - Sound "synth": closest existing pack → chill/summer electronic vibe
  - Card .elevated: as explicitly requested
  - Glow 0.20: within the 0.14-0.26 window, balanced
